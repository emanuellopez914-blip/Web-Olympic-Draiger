¿Cómo usar este menú?

  1. Selecciona un examen

    Elige entre los exámenes pasados para ver sus resultados, fotos y diplomas.

  2. Ver resultados completos

    Haz clic en "Ver Resultados" para acceder a la página detallada del examen seleccionado.

  3. Genera código QR específico

    Usa "QR Específico" para crear un código que lleve directamente a ese examen.

  4. Consulta próximas fechas

    Revisa el calendario de exámenes futuros para planificar tu preparación.
